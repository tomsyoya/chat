# chat
- Node.jsを使ってチャットアプリを作成します。  
- Slackのようなチャットアプリを作成するのが目標です。  
- 皆で勉強できるようなプロジェクトなので参加歓迎します。  

## develop
### clone
```sh
git clone https://github.com/minmin0530/chat.git
cd chat
```

### setup
```sh
npm install
```

### run
```sh
node index.js
```
